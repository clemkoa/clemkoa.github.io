# clemkoa.github.io

[https://clemkoa.github.io](https://clemkoa.github.io)

Just a static website for random blog posts.
Based on Jekyll https://jekyllrb.com/docs/.

## Installation (macOS)

1. Install `brew`
2. run

```bash
brew install ruby
gem install jekyll bundler
bundle add webrick
bundle update
bundle install
```

## Dev

```
bundle exec jekyll serve --livereload --drafts
```
